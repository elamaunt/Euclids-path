# Фрактал Евклида — визуализация объектов twin-программы

`euclid_fractal.py` рисует структуру простых через **реальные объекты** программы
(центры `6m±1`, ранги, old-peel спуск, двигатель) — никакой декоративной генерации,
каждый вид — прямая отрисовка арифметики. Выход: PNG в `out/` (английские подписи,
colorbar/легенды/оси).

```sh
python euclid_fractal.py    # сгенерировать все виды заново
```

Зависимости: `numpy`, `matplotlib`.

## Виды

| # | Файл | Что нарисовано |
|---|---|---|
| 1 | `1_old_peel_tree.png` | **Лес old-peel спуска** — фрактал двигателя: рекурсивный peel `6n∓ε = p·(6t±δ)` от полосы стартовых центров; цвет = глубина спуска |
| 2 | `2_rank_field.png` | **Поле ранга/заряда** `r₋(m) − r₊(m)` по центрам — CRT-рельеф слоя простых `(A, A²]` |
| 3 | `3_twin_spiral.png` | **Золотая спираль центров** (угол — золотой) — близнецы как золотые двойники |
| 4 | `4_descent_landscape.png` | **Ландшафт нагрузки** `Ω(6m−1)+Ω(6m+1)` по малым простым — тёмные долины = кандидаты в близнецы |
| 5 | `5_twin_line_genealogy.png` | **Линия центров**: близнецы — золотые двойные искры **по обе стороны** линии (`6m+1` сверху, `6m−1` снизу); **родословные** old-peel — дуги **вдоль самой линии**, цвет дуги = **простое Евклида** шага `6m∓1 = p·(6t±1)` |
| 6 | `6_genealogy_ornament.png` | **Узор из родословных**: полный old-peel спуск каждого центра — хорды на круге центров (Безье к середине), цвет = простое Евклида шага; CRT-периодичность малых простых ткёт розетку; близнецы (пустая родословная — нечего чистить) — золотой обод |

Виды 5–6 — прямая иллюстрация финальной архитектуры узла: генеалогии
(`ExtendedProperGeneratedFlow`) живут **на** линии центров, близнецы —
терминальные объекты **вне** её (у twin-центра нет собственного peel:
обе стороны просты — ср. заплату `properDiv` в `ConcreteStep00Graph`).
