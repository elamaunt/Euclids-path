## Цель шага

Зафиксировать **что** доказывается, **через что** и в **каком порядке**, и честно обозначить, какие
звенья уже строгие, а какие — открыты. Это оглавление всей цепочки [[01]]–[[15]].

## Главная теорема (цель)

**Гипотеза простых-близнецов.** Существует бесконечно много простых `p`, для которых `p+2` также
простое. Эквивалентно: существует бесконечно много натуральных `m`, для которых одновременно простые
$$6m-1,\qquad 6m+1.$$

Эквивалентность форм — содержание шага [[01]].

## Стратегия (10 слоёв)

1. **Центровая редукция** [[01]]. Всякая нетривиальная пара близнецов записывается как `6m−1, 6m+1`.
2. **CRT-слой запретов** [[02]]. Каждый простой `q≥5` запрещает ровно два класса индекса:
   `m ≡ ±6⁻¹ (mod q)`.
3. **Активное просеивание** [[03]]. Если `m` избегает всех активных запретов до корня — оба числа
   простые.
4. **Rank-бухгалтерия** [[04]]. Вводятся side-ranks `r₋(m), r₊(m)` и классы `N_{ij}`; главная величина
   $$B_5 := N_{00}-N_{33}.$$
   Достаточно доказать `N_{33} < N_{00}`.
5. **Детерминантный граф** [[05]]. Для «плохого» слоя rank-(3,3): `6m−1=abv`, `6m+1=qrs`, откуда
   $$abv-qrs=-2.$$
6. **Вечный двигатель Евклида** [[06]] — ключевой объект. Невозможность бесконечной самоподобной
   цепи спусков (гармоническая высота строго убывает; целочисленный центр не может падать ниже 1).
7. **Charged-ledger** [[07]]. Все структурные исключения попадают в конечный заряженный слой:
   fixed, high-gcd, shared CRT, affine/projection lock, repeated chord, bounded cycle.
8. **Terminal reservoir / O4C-res** [[09]], [[10]]. Покрытие сертификатов оплачивается terminal-рядами:
   $$\mathcal M_{03}+\mathcal M_{30}\ \ll\ \frac{N_{03}N_{30}}{N_{00}}+P^\varepsilon.$$
9. **DASC и G2** [[11]], [[12]]. Верхняя нормировка product-corner и genuine determinant gap:
   $$N_{33}\ \le\ \frac{N_{03}N_{30}}{N_{00}}-c_G\frac MA+\eta_B\frac MA+o(M/A).$$
10. **Финальная сборка** [[13]], [[14]]. При `c_G>η_D+η_B` получаем `N_{33}<N_{00} ⇒ B_5>0 ⇒`
    существует выживший центр ⇒ пара близнецов; итерация блоков ⇒ бесконечность.

## Сжатый dependency-graph

```
CRT/active layer ─▶ rank decomposition ─▶ B₅ = N₀₀ − N₃₃
N₃₃ ─▶ abv−qrs=−2 ─▶ Cb−Dr=−2 ─▶ inverse slot
        ─▶ charged ledger (Bad ⊂ Charged_{≤K} ∪ O(Pᵉ))
        ─▶ certificate cover ─▶ terminal supports (ℳ₀₃+ℳ₃₀)
        ─▶ O4C-res ─▶ N₀₃N₃₀/N₀₀
anchor-excess ─▶ DASC ─▶ N₀₃N₃₀/N₀₀ ≤ N₀₀ + η_D·M/A + o
determinant gap ─▶ G2 ─▶ N₃₃ ≤ N₀₃N₃₀/N₀₀ − c_G·M/A + η_B·M/A + o
DASC + G2 + (c_G > η_D + η_B) ─▶ N₃₃ < N₀₀ ─▶ B₅ > 0 ─▶ twin primes
```

## Глоссарий обозначений

- `m` — индекс центра пары `6m−1, 6m+1`.
- `A` — нижняя граница активного слоя; основной слой простых-clock — `(A, A²]`.
- `X ≍ A⁴` — масштаб dyadic-блока.
- `r₋(m), r₊(m)` — side-ranks (число простых из `(A,A²]`, делящих `6m∓1`).
- `N_{ij}` — число центров с рангами `(i,j)`; `B₅ = N₀₀ − N₃₃`.
- `M` — масштаб массы «плохого»/carrier-слоя; `P` — локальный product-параметр, `a,v,q,s ≍ P`.
- `C=av, D=qs` — product-переменные; `ℳ₀₃, ℳ₃₀` — terminal-row reservoirs.
- `η_D, η_B` — константы DASC/payment; `c_G>0` — константа determinant-gap.
