# 42. Ходж: квантованные заряды и оплата циклами

> Модули: `Engine/HodgeFront.lean` (зелёная цепь + трилемма, всё 🟢),
> `Engine/CausalClosureAxiom.lean` §14 (карантин: одна растяжка, 🟡).
> Статусы: 🟢 = машинно, стандартные аксиомы; 🟡 = AXIOM-TAINTED; 🔴 = открытый вход.

## Прочтение

**Класс Ходжа = квантованный (рациональный) заряд когомологий выровненного
типа (p,p). Алгебраический цикл = оплата. Гипотеза = всякий квантованный
выровненный заряд оплачен.** Неоплаченный заряд открыл бы бесконечный строгий
спуск по ℕ-высоте (знаменателю/сложности остатка) — запрещённый вечный
двигатель (EPMI).

## Честная граница (громко)

В mathlib теории Ходжа НЕТ — проверено по пину дословно: слово «Hodge»
встречается ровно трижды: цитата книги W. Hodges по теории **моделей**
(`ModelTheory/Fraisse`) и упоминания p-адической теории Ходжа в комментариях
периодных колец (`RingTheory/Perfectoid/BDeRham`, `FontaineTheta`);
hodge-star-оператора тоже нет. `HodgeLedger` — абстрактный леджер: классы,
`IsHodge`, `IsAlgebraic`, `height : Cls → ℕ` и якорь квантования
«оплачен ⟺ высота 0» (потреблён по-настоящему — в кванте неоплаченности L1 и
в V1-опровержении). Никакого содержания алгебраической геометрии сверх
намеренной будущей инстанциации; **проблема тысячелетия НЕ решается и НЕ
объявляется**; остаток — per-model 🔴 вход `DescentLaw`.

## Двигатель мёртв безусловно (заголовочная новость ветви)

`UnpaidDescentChain` — бесконечная строго убывающая ℕ-цепь неоплаченных
зарядов — и **`isEmpty_unpaidDescentChain`** 🟢: цепей не существует НИ В
ОДНОЙ модели, без единой гипотезы (EPMI, A = 1; список аксиом — только
`[propext, Quot.sound]`, даже без выбора). **Раскрытая асимметрия с
Янгом–Миллсом**: там ℝ-лестница жила ровно при безмассовости
(`gapless_iff_nonempty_ladder`) и убивалась декретируемым законом
квантования; здесь квантование ВСТРОЕНО в модель (`height : ℕ`) — двигатель
мёртв всегда, и вся содержательность ветви сместилась в существование
ШАГОВ спуска, т.е. в закон.

## Герой и коллапс зеркала стоимости

- **Герой** `hodgeProperty_of_descentLaw` 🟢 — закон спуска ⟹ гипотеза
  Ходжа модели: сильная индукция по высоте (выбор не нужен); цепной маршрут
  через choose (`descentSeq`, зеркало ladderSeq ЯМ) дан второй формой.
- `no_unpaid_lawful_model` 🟢 — модели «закон + неоплаченный заряд» не бывает.
- **Обратная сторона ВАКУУМНА** (`descentLaw_of_hodgeProperty` — квантор по
  неоплаченным пуст; стиль римановской L5, в отличие от ЯМ, где ранг
  строился по-настоящему) — раскрыто.
- **Коллапс** `descentLaw_iff_hodgeProperty` 🟢 — per-model закон ⟺ гипотеза,
  зелёно и без всякой границы: форма осуждённого моста. Декретировать закон
  для модели = декретировать её гипотезу дословно. Потому шестого поля нет;
  недостающее — data anchor (настоящие рациональные (p,p)-классы).

## Трилемма шестой границы: поля НЕТ

- **V1** `hodgeLawUniversal_refuted` 🟢 — универсальная форма опровержима:
  кованая модель `cookedUnpaid` (классы ℕ, ходжевы {0,1}, оплачен только 0,
  высота тождественная) несёт неоплаченный заряд высоты 1, а шаг спуска
  упёрся бы в оплаченность нулевой высоты (якорь квантования). Декрет
  взорвал бы карантин.
- **V2** `hodgeLawExistential_green` — экзистенциальная форма уже доказана
  (`cookedPaid`: закон вакуумен, гипотеза дословно) — и зависит **ни от
  каких аксиом вообще**.
- **V2′ — уникальная структурная точка ветви**:
  `hodgeChainManifestationLaw_green` 🟢 — chain-манифестационная форма
  (механический перенос ЯМ-кандидата с лестниц на цепи) **вырождена в
  зелёную теорему**: цепей нет ни в одной модели, закон над ними вакуумен.
  Вердикт V2-типа, НЕ V3 — из пустого типа свидетелей V3 не собрать.
- **V3** `hodgeManifestationLaw_refutes_boundary` 🟢 — манифестационная
  форма над ПРЕДЪЯВИМЫМ свидетелем (одиночный неоплаченный класс,
  `cookedUnpaidClass` — конструкцией, без выбора) несовместима с принятой
  границей; характеризация — «⟺ глобальная заморозка всех леджеров»,
  безусловная.

Аудиты §7 добивают: **A1′** — связка Bridge∧Impossible ⟺ РОВНО гипотеза
Ходжа модели (`front_pair_iff_RH` при RH := HodgeProperty); связка
опровергнута на кованой модели; FreeLawCollapse + ¬ZeroSeparating.

## Карантин §14

Единственное честное 🟡 — растяжка
`quarantine_inconsistent_if_hodgeManifestationLaw_decreed`: декрет
манифестационной формы взорвал бы карантин ровно здесь. Безопасность
проверена: закон ⟺ глобальной заморозке, а зелёный мир знает заморозку
только при A ≤ 4 — масштабы A ≥ 5 подлинно открыты.

## Место в общем ходе

Шестой фронт одной машины (близнецы → Риман → P/NP → Янг–Миллс → НС → Ходж),
один корень — невозможность вечного двигателя. Серия трилемм устойчива:
честные декретные поля существуют только там, где свидетель отклонения
зелёно непредъявим (twin-узел, off-critical нуль); всюду, где отклонение
куётся (фреймы, спектры, профили, неоплаченные заряды), — декрет либо
взрывается, либо вакуумен, и остаётся зелёная условная теорема с честно
именованным входом. У Ходжа к этому добавилась собственная новость:
квантованность модели делает двигатель мёртвым безусловно — гипотеза
эквивалентна существованию самих шагов оплаты.
