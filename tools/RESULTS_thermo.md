# Термодинамика: Cov = CROSS - DIAG (два закона)

Скрипт: thermo_harness.py.

Cov(r-,r+) = CROSS - DIAG, где
  DIAG = sum_p P_p^- P_p^+   (теплота, убираемая эксклюзивностью = 2-й закон / no-backward)
  CROSS = sum_{p!=q}(joint - product)   (граничная CRT-ошибка для различных простых)

ВЫВОД: знак Cov<0 форсируется DIAG (эксклюзивностью / 2-м законом) во ВСЕХ случаях; |CROSS|<DIAG везде.
Граничная ошибка субдоминантна (минимум ~5% при κ=4.4). MI->0 = равновесие/независимость (ранг-1).

Два закона:
  1-й (сохранение двойки): TwoGap (XY-ZW=2), формализовано.
  2-й (необратимость, no-backward): NoBackward (диагональ исчезает), формализовано => DIAG>0 => Cov<0.
Остаток для доказательства знака: |CROSS| <= DIAG (сито-оценка теплоты, НЕ Kloosterman).

## ОБНОВЛЕНИЕ (k до 28): чистый закон теплоты
DIAG * A * ln A ≈ 0.80–0.85 (постоянна) => DIAG ≈ 0.81/(A ln A). Стабильно k=25..28, κ=4 и 4.4.
CROSS/DIAG не расходится, при k=28 мал (0.069/0.123). cornerD<0 всюду. R_fc->~0.978.
Главная цель: доказать DIAG - |CROSS| > 0 всюду (= four-corner sign). Нужен закон CROSS (o(DIAG)?).
