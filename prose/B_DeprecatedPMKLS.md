# B. История: снятый PMKLS-слой

> Источник: base/UPDATED §9, §XIX–XXI; AUDIT_REPLACED §59, §61–62, §67; BE_UPDATED §XXVI–XXVII.
> Lean-аналога нет (историческая справка).

## Зачем этот файл

Чтобы при чтении исходников **не обмануться миграцией статусов**: один и тот же блок (PMKLS) сначала
развивается как опора и даже «доказывается внутренним аргументом», а затем поздним аудитом **снимается**
как недоказанный. Итоговый статус задают последние файлы.

## Что такое PMKLS

**Product-Modulus Kloosterman Large Sieve** — спектральная оценка вида
$$\sum_{C}\Big|\sum_{D}\beta_D\,e_C(h\overline D)\Big|^2\ \ll\ P^{2+o(1)}\sum_D|\beta_D|^2,$$
которой в ранних версиях нормировался inverse-slot из [[05]] (цепь
`PMKLS ⇒ PF-LD ⇒ SID ⇒ VMSR ⇒ bounded certificates`).

## Хронология статуса

1. **base** (`…assembly_…30.md`): PMKLS — рабочая опора, Теорема 9.1.
2. **UPDATED**: оговорка — это **Structured PMKLS** (внутренний inverse-large-sieve из §XIX–XXI), а не
   импорт общей опубликованной Kloosterman large sieve для произвольных коэффициентов.
3. **AUDIT_REPLACED** (§62): «прежний блок Structured PMKLS/QSPMKLS больше не считается доказанным и
   удаляется из финальной логики»; arbitrary-coefficient Kloosterman-fraction large sieve признан
   избыточным и недоказанным в нужной силе. Замена — короткий **CRT row-mixing** при `P⁴=o(M/A)`
   (т.е. `P≤A^{3/4−ε}`); остатки `V₁–V₄` [[15]].
4. **BE_UPDATED** (§XXVI–XXVII): row-mixing оставлен как полезная локальная замена, но **не** как
   единственная финальная опора; финальная форма остатка переведена в детерминированный
   **clean descent / Boundary-Exit** слой [[06]]; центральный остаток — BDNC⁺ [[15]].

## Вывод

Линия опоры: **PMKLS/QSPMKLS → CRT row-mixing → clean descent / Boundary-Exit**. Разделы PMKLS в
исходных сборках сохранены как исторические; в консолидированной цепочке [[00]]–[[14]] они **не**
используются как финальное закрытие. Поэтому при сверке с исходниками статус «доказано» нужно брать из
последней редакции (BE_UPDATED), а не из ранних.
