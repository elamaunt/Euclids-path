# 05. Детерминантный закон rank-(3,3)

> Lean-аналог: `EuclidsPath/Step05_DeterminantLaw.lean` (заготовка)
> Источник: BE_UPDATED §7–8 (Теорема 7.1, Лемма 8.1).

## Цель шага

Перевести «плохую клетку» `N₃₃` в точное диофантово уравнение и показать его жёсткость (zero-one
slot), на которой держатся все верхние оценки [[09]]–[[12]].

## Определения / формулировки

Для `m ∈ N₃₃` оба числа раскладываются в произведение трёх простых из слоя `(A,A²]`:
$$6m-1=abv,\qquad 6m+1=qrs,\qquad a\le b\le v,\ q\le r\le s.$$

**Теорема 7.1 (детерминантный закон).** Вычитая, получаем точное уравнение
$$abv-qrs=-2.$$
Обратно, каждое допустимое решение задаёт центр `m=(abv+1)/6=(qrs-1)/6` при выполнении
congruence-admissibility.

Вводя product-переменные `C=av`, `D=qs`, уравнение принимает вид
$$Cb-Dr=-2,\qquad\text{откуда}\qquad b\equiv -2\,\overline C\pmod D\ \ (\text{при }(C,D)=1).$$

**Лемма 8.1 (zero-one slot).** В допустимом balanced-боксе для фиксированной четвёрки `(a,v,q,s)`,
т.е. для фиксированных `(C,D)`, существует **не более одного** допустимого `b`.

## Идея доказательства

`qrs − abv = 2`. Решения `b` лежат в одном классе по модулю `D=qs > A²`, а `b ∈ (A,A²]` имеет длину
интервала меньше модуля — значит не более одного представителя.

## Статус

🟢 **строго.** Линейная алгебра + оценка длины интервала против модуля.
