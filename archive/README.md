# archive/ — изучено по пути, обойдено

Здесь сохранена **первая (аналитическая) линия** доказательства, чтобы был виден ход мысли.
Она **обойдена** структурной Engine-линией (см. `prose/00_Overview.md`).

## Что это
Детерминированная Euclidean–CRT программа через **аналитический ledger**:
PMKLS / Kloosterman / large-sieve / DASC / G2 / O4C-res / BDNC⁺, с центральной величиной
`B₅ = N₀₀ − N₃₃` и попыткой замкнуть `N₃₃ < N₀₀` через determinant-gap.

## Почему обойдена
Её несущие мосты (DASC, G2, неравенство `c_G > η_D + η_B`, O4C-res) остались открытыми и по
сложности сопоставимы с самой гипотезой — это аналитика/распределение (красная линия программы).
Engine-линия заменила их **структурным** Multi-Rank Non-Cover + законами двигателя, сведя всё к
одному не-счётному узлу **SNOL**.
