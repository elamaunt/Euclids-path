# Euclids-path

Чистая консолидация доказательной программы гипотезы **простых-близнецов**, выстроенной вокруг
**вечного двигателя Евклида** (принцип невозможности бесконечного «чистого» спуска — версия
бесконечного спуска Ферма).

Это **не новое доказательство** и **не завершённое доказательство**. Это попытка собрать
разрозненные рабочие записи (несколько параллельных подходов и версий «полной сборки») в **единую
линейную, пронумерованную цепочку рассуждений** — так, чтобы было видно:

- из чего состоит каркас и как одно следует из другого;
- что строго доказано, а что пока только сведено к открытой лемме;
- где именно проходит реальная граница известного.

> **Источник истины.** Первичные записи лежат в `f:/Primes/*.md` и `f:/Primes/*.csv` (16 файлов,
> 28–30 июня 2026). Они **не редактируются и не удаляются**. Этот репозиторий — производная
> консолидация: при любом расхождении первичны исходники.

## Легенда статусов

В прозе у каждой леммы/теоремы стоит метка:

- 🟢 **строго** — доказано элементарно/классически, без скрытых допущений;
- 🟡 **локально** — доказана локальная/структурная часть, но она не закрывает глобальный вывод;
- 🔴 **ОТКРЫТО** — сведено к недоказанной лемме; по сложности может быть сопоставимо с самой гипотезой.

В Lean тот же смысл выражен механически:

- теорема **без** `sorry` — реально проверяемая (после компиляции) часть;
- теорема с `sorry` и комментарием `-- OPEN:` — открытое место.

Это главный смысл параллельного Lean-слоя: **`sorry` нельзя подделать**, поэтому формализация не даёт
выдать недоказанное за доказанное.

## Краткая суть программы

Центр близнецов `6m±1` → CRT-запреты `m≡±6⁻¹ (mod q)` → ранги и `B₅=N₀₀−N₃₃` → детерминант
`abv−qrs=−2` → charged-ledger `Bad ⊂ Charged_{≤K} ∪ O(Pᵉ)` → terminal/O4C-res → **DASC + G2** →
при `c_G>η_D+η_B` выходит `N₃₃<N₀₀ ⇒ B₅>0 ⇒` существует выживший центр `m` ⇒ пара близнецов;
итерация блоков `P→∞` ⇒ бесконечно много пар.

**Вечный двигатель Евклида** — гипотетическая бесконечная самоподобная цепь `XᵢYᵢ−ZᵢWᵢ=2`,
сохраняющая положительную массу без потерь; её существование равносильно отсутствию близнецов.
Программа доказывает **невозможность** такого двигателя (строго — локально; глобальная версия пока
открыта).

**Численно** (`twin_stats_*.csv`): `N₃₃/N₀₀ ≈ 0.011–0.016` (порядка 1–1.6 %) на `k=21…28`,
`Γ/M ≈ 0.67→0.70` — `B₅>0` выполняется с большим запасом. Это сильная эмпирика, не доказательство.
Подробная таблица — [приложение A](prose/A_NumericalEvidence.md).
